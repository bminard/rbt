Development

Versioning
++++++++++

Releases follow the `Semantic Versioning`_ guidelines for version numbers.

Contributing
============

Coding Style
++++++++++++

Please follow the `Google Python Sytle Guide`_ for contributions.

See :file:`virtualenv.sh`.

Requires `virtualenv`_.
